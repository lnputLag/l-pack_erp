
# L-Pack ERP Client

Система планирования ресурсов предприятия, клиентская программа.

После сборки, скопируйте в директорию с выполняемыми файлами
(Client/bin/Debug|Release)
конфигурационный файл:

`_application.config`

убрав символ подчеркивания и отредактировав содержимое.

Стартовый проект внутри solution: Client

## Описание директорий

CodeReason.Reports -- библиотека генерации отчетов
Lib -- внешние библиотеки, поставляемые в бинарном виде
